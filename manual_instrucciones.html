Darwin User Manual

Introduction

This virtual assistant helps you to find information, play songs, opening apps and more using simple voice or text commands.

Getting Started

1. Activation

Choose the lenguage that you want to talk with the assistant.

2. Basic Commands

Speak with keywords like "Open Chrome", but you can also talk naturally. For example: "What time is it?," "How are you?"

Key Features

1. Ask Questions

  • Searching Features: "Search who is Steve Jobs"
  • Use your device: "Open Chrome"
  • Computer Status: "I'm connected to the wifi?"

2. Entertainment

  • Play music: "Play a song."
  • Tell jokes: "Tell me a joke."

Tips for Best Use

  • Speak clearly and in a quiet environment.
  • Use precise commands for faster responses.

Troubleshooting

  • Ensure the microphone is not muted.
  • Restart the device if issues persist.
